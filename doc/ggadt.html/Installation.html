2 Installation

2.1 System requirements

GGADT should install easily on most Unix-based systems (e.g. Linux, Mac) by following the standard GNU install procedures, described in INSTALL.

./configure [--enable-openmp] ... make make install

2.2 Package dependencies

  • FFTW 3+
    • If you wish to use the OpenMP variant of GGADT, you must have the fftw3_omp libraries installed
    • In order for GGADT to find the Fortran headers for FFTW3, they must be in the standard location (/usr/local/include or usr/include) or in $prefix/include where $prefix is defined by doing ./configure --prefix=<prefix> ....
  • For the OpenMP version of GGADT, you must have C and Fortran compilers that are compatible with OpenMP 4.0. This version of GGADT was tested with the GNU C and Fortran compilers (gcc, gfortran) version 4.9. Attention Mac users: You must have the command line tools from XCode installed, but beware of your supplied version of gcc – it is NOT going to work for compiling the OpenMP version of GGADT. The version of GCC that you get with XCode is a front-end for the clang compiler and is quite dated. We recommend obtaining later versions of GNU compilers, which you can find here: http://hpc.sourceforge.net/.
  • Python >=2.7 for some of the included scripts.
  • matplotlib python library if you wish to use the included plotting scripts.
