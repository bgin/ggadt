1.1 Language

GGADT is written in Fortran, using the Fortran 2003 standard. In the future, GGADT may be changed to comply with earlier Fortran standards (90 or 95) for compatibility reasons. Most importantly, this will allow for easier implementation of an MPI version of GGADT.

Fortran 2003 provides an easy way to interface Fortran programs with C libraries. In this case, Fortran 2003 made the FFTW3 library (http://www.fftw.org/) easy to implement for performing fast Fourier transforms. FFTW3 is a well-tested, optimized, and popular suite for performing FFT's.
